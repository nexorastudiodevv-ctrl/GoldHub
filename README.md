# 🌟 Gold & Currency Hub - موقع تتبع الذهب والعملات

## 📋 نظرة عامة

موقع ويب متقدم ومتخصص لمتابعة **أسعار الذهب العالمية** و**أسعار الصرف** للعملات الرئيسية في الوقت الفعلي، مع واجهة مستخدم حديثة وميزات احترافية.

---

## ✨ الميزات الرئيسية

### 💰 تتبع الذهب الحقيقي
- أسعار الذهب العالمية المحدثة كل 5 ثوان
- حساب سعر الجرام بثلاث عيارات (24K, 21K, 18K)
- عرض الأسعار بالعملة المحلية (جنيه مصري)
- رسم بياني مباشر لتتبع التغيرات (آخر 20 نقطة)

### 💱 محول العملات الفوري
- تحويل فوري بين الدولار والعملات الأخرى
- أسعار صرف محدثة لحظياً
- دعم: USD, EUR, EGP, SAR وغيرها
- حساب تلقائي مع كل كتابة

### 🔔 الإشعارات الذكية
- تنبيهات برمجية عند الوصول لسعر معين
- اختيار من عدة نغمات تنبيه
- حفظ الإشعارات محلياً على الجهاز
- دعم Desktop notifications

### 📊 الرسوم البيانية المتقدمة
- رسم بياني حي باستخدام Chart.js
- عرض تاريخ الأسعار بصريًا
- تحديث تلقائي كل تغيير سعر
- ألوان عصرية وواجهة احترافية

### 📱 تطبيق ويب تقدمي (PWA)
- يمكن تثبيته على الهاتف كتطبيق حقيقي
- يعمل بدون إنترنت (وظائف محدودة)
- واجهة سلسة وسريعة جداً
- حجم تنزيل صغير جداً

### 🎛️ لوحة تحكم إدارية
- تحديث الأسعار يدويًا عند الحاجة
- مزامنة فورية مع Firebase
- تعديل مصاريف التصنيع
- سجل تحديثات الأسعار

### 🌍 تحديد الموقع التلقائي
- كشف العملة حسب موقع الجهاز
- تحديث اللغة تلقائياً
- تذكر الإعدادات المفضلة

### 🎨 واجهة عصرية احترافية
- ديزاين Glassmorphism أنيق
- ألوان نيون سماوية وخضراء
- واجهة داكنة (Dark Mode)
- تصميم متجاوب (Desktop, Tablet, Mobile)

## 🛡️ أخطاء تم إصلاحها

### ✅ الخطأ الأول: manifest.json
```diff
- "purpose": "any maskable"  ❌ (صيغة خاطئة)
+ "purpose": "any"           ✅ (صيغة صحيحة)
```

### ✅ الخطأ الثاني: sw.js
- ❌ قوس إغلاق مكرر في النهاية
- ❌ معالجة أخطاء ناقصة
- ❌ استراتيجية تخزين مؤقت غير محسّنة

**الحل:**
- ✅ إزالة القوس المكرر
- ✅ إضافة معالج activate لتنظيف الـ Cache
- ✅ تحسين استراتيجية network-first
- ✅ إضافة معالج أخطاء عام
- ✅ إضافة self.skipWaiting() للتحديث الفوري

## 🐛 الدعم والمشاكل الشائعة

### المشكلة: الأسعار لا تتحدث
```
الحل:
1. تحقق من الإنترنت
2. افتح Console (F12) وابحث عن الأخطاء
3. أعد تحميل الصفحة (Ctrl+R)
```

### المشكلة: الإشعارات لا تظهر
```
الحل:
1. تفعيل الإشعارات في إعدادات المتصفح
2. تأكد من الصوت مفعل
3. افحص الـ Application tab في DevTools
```

### المشكلة: Service Worker لا يعمل
```
الحل:
1. استخدم HTTPS أو localhost فقط (لا file://)
2. تحقق من Application → Service Workers في DevTools
3. امسح الـ Cache وأعد التحميل
```
