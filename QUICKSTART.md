## 🚀 الطريقة 1: فتح المباشر (الأسرع)

### على Windows:
1. انقر مزدوج على `index.html`
2. سيفتح في متصفحك الافتراضي

### على Mac:
1. انقر على `index.html` بزر الفأرة الأيمن
2. اختر "Open with" → اختر متصفح

### على Linux:
```bash
xdg-open index.html
```

---

## 🚀 الطريقة 2: استخدام خادم محلي (الموصى به)

هذه الطريقة أفضل لأن Service Worker يعمل بشكل صحيح فقط مع HTTPS أو localhost.

### باستخدام Python:
```bash
# في مجلد المشروع (GOOD)

# Python 3.x
python -m http.server 8000

# أو Python 2.x
python -m SimpleHTTPServer 8000
```

ثم افتح في المتصفح:
```
http://localhost:8000
```

### باستخدام Node.js:
```bash
# تثبيت http-server مرة واحدة
npm install -g http-server

# تشغيل الخادم
http-server

# سيظهر رابط محلي
```

### باستخدام Live Server (VS Code):

1. فتح المشروع في VS Code
2. انقر اليمين على `index.html`
3. اختر "Open with Live Server"

## 🔍 تصحيح المشاكل السريع

### المشكلة: الأسعار لا تظهر

**الخطوة 1:** افتح Developer Tools
```
Windows: Ctrl + Shift + I (أو F12)
Mac: Cmd + Option + I
```

**الخطوة 2:** انظر إلى Console (علامة البرمجة)
- هل تظهر أي أخطاء حمراء؟
- اكتب: `console.log(goldPrice)` واضغط Enter

**الخطوة 3:** تحقق من الإنترنت
- هل الاتصال يعمل؟
- جرّب فتح موقع آخر

**الحل السريع:** أعد تحميل الصفحة (Ctrl+R أو Cmd+R)

---

### المشكلة: Notifications لا تظهر

**التحقق:**
```
Windows: إعدادات > الإشعارات
Mac: System Preferences > Notifications
```

**الحل:**
1. تأكد من تفعيل الإشعارات للمتصفح
2. تأكد من الصوت مفعل
3. جرّب على صفحة أخرى أولاً

---

### المشكلة: الموقع بطيء

**الحلول:**
1. امسح الـ Cache
   ```
   Ctrl+Shift+Delete > "All time" > حذف
   ```

2. أعد تحميل بدون Cache
   ```
   Ctrl+F5 (أو Cmd+Shift+R على Mac)
   ```

3. أغلق التبويبات الأخرى

## 💡 نصائح مفيدة

### 📌 السرعة:
- استخدم `localhost` بدل `file://`
- استخدم Service Worker المدمج
- Data يُحفظ محلياً

### 🔐 الأمان:
- كل بيانات تُرسل عبر HTTPS (في الإنتاج)
- لا توجد بيانات شخصية تُجمع
- التسجيل الدخول آمن

### 📊 الأداء:
- الرسم البياني محسّن
- API requests محدودة
- Caching فعّال
